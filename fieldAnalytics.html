Data Engineering

Data Engineering is software development focused on data: data sources, data modeling and data "plumbing".

Field Analytics leverages API's, open source software and cloud technology to move data between isolated systems. This gives business users access to the information they need.

Example data flows:

  • Accounting & ERP systems (Great Plains/Dynamics, NetSuite, SAP, Sage, QuickBooks, etc.)
  • Back-end operational systems
  • Web-based software applications (Sales Force, Mail Chimp, etc.)
  • Emails
  • Spreadsheets
  • Databases
  • Data scraped from websites
  • Internal security systems (Active Directory, LDAP, etc.)

Machine Learning

Machine learning uses algorithms to uncover hidden insights through learning, historical relationships and trends in data.

Field Analytics builds predictive analytical models so that your team can make better decisions to help you compete with large organizations utilizing big data.

Example use cases:

  • Credit decisions / fraud risk management
  • Demand planning / sales forecasting
  • Customer segmentation / offer targeting
  • Customer churn and retention analysis
