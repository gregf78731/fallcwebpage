Data Engineering

Field Analytics leverages API's, open source software and cloud technology to move data between isolated systems. This gives business users access to the information they need.

Enables access to data in accounting systems (Great Plains, QuickBooks, etc.), software applications (Sales Force, Mail Chimp and other marketing systems), emails, spreadsheets, databases, data scraped from websites, Active Directory, etc., etc.

Areas of expertise include:

  • Python, SciKit-Learn, Pandas, NumPy, Matplotlib
  • APIs / Web Services using JSON or SOAP/XML
  • Amazon Web Services (Redshift, RDS, Lambda, API Gateway, EC2, Boto)
  • Databases (Oracle, Microsoft SQL Server, MySQL, PostgreSQL)
  • Excel pivot tables and OLAP data cubes
